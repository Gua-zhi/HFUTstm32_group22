# 有关stm32内容
键盘控制项目:当按下第n个键盘时，数码管显示数字n且灯亮
学号姓名滚动
外壳文件内容
考虑使用蓝牙和QR在同一个模式共存的形式完成任务，仅仅采取了模式的拼接
融合了蓝牙和蓝牙调整pid的模式，结合k210视觉模块，对QR二维码进行识别（颜色判断太不准确）
目前，能做到K210先学习三种颜色的小球，记录下阈值，然后识别相应颜色的数量
stm32开发板通过TX和RX连接传递信息到51单片机上，51单片机的液晶屏显示RGY三种颜色的数量

通过所给代码将蓝牙模块k210视觉识别二维码结合起来，做到小车在一直开机的情况下，可以利用qrcode进行移动，同时手机蓝牙可以操控，另外发现单独引入蓝牙控制并不能pid调控，所以又加了pid调整的适配。
准备利用qrcode实现对小车的引导运动，k210在学习识别红黄绿三种颜色小球后，可以大体上识别出颜色个数，并利用串口2传递给stm32，之后stm32通过串口3传给51开发版，使得51开发板可以接受信息，在液晶屏上显示red green yellow三种小球颜色的数量(R: G: Y:)
同时带有#$Y表示stm数据向小车传输，表示信息流正在传输
